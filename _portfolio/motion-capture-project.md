## Project Overview

This project involved the development of motion capture algorithms and Android applications for sports assessment and performance analysis. Working under Prof. James Cheng at CUHK, I contributed to creating innovative solutions for analyzing athletic movement patterns.

## Key Features

- **Motion Capture Algorithms**: Advanced algorithms for tracking and analyzing movement
- **Android Development**: Mobile application for real-time motion data analysis
- **Sports Assessment**: Performance evaluation tools for athletes and coaches
- **Real-time Processing**: Efficient algorithms for live motion analysis

## My Contributions

- Contributed to the design and implementation of motion capture algorithms
- Participated in Android application development for motion data visualization
- Optimized algorithms for real-time performance on mobile devices
- Collaborated with research team on sports assessment methodologies

## Technical Stack

- **Programming**: Java (Android), Python (algorithms), C++ (performance optimization)
- **Mobile Development**: Android SDK, camera APIs, sensor integration
- **Computer Vision**: OpenCV, motion tracking algorithms
- **Data Processing**: Real-time signal processing, statistical analysis

## Research Period

**Duration**: March 2024 – August 2024  
**Supervisor**: Prof. James Cheng  
**Institution**: The Chinese University of Hong Kong  
**Role**: Research Assistant

## Applications

This technology has potential applications in:
- Athletic performance optimization
- Injury prevention and rehabilitation
- Sports coaching and training
- Biomechanical analysis
- Movement disorder assessment

## Impact

The project demonstrates the practical application of computer vision and mobile computing in sports science, providing valuable tools for athletes, coaches, and researchers to better understand and improve human movement patterns.
