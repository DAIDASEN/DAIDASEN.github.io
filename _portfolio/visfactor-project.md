## Project Overview

VisFactor is a novel benchmark designed to evaluate the fundamental visual cognition capabilities of Multimodal Large Language Models (MLLMs). This project represents a significant contribution to understanding how AI systems process and reason about visual information.

## Key Features

- **Comprehensive Evaluation Framework**: Based on the Kit of Factor-Referenced Cognitive Tests
- **Systematic Analysis**: Identifies current challenges faced by MLLMs in visual reasoning
- **Research Impact**: Published as arXiv:2502.16435 (2025)

## My Contributions

- Developed benchmark architecture using the Kit of Factor-Referenced Cognitive Tests
- Implemented core evaluation algorithms for assessing MLLM capabilities
- Conducted comprehensive analysis of model limitations and performance patterns
- Co-authored the research paper with detailed findings and insights

## Technical Stack

- **Programming**: Python, Machine Learning frameworks
- **Research Tools**: Statistical analysis, cognitive testing frameworks
- **Evaluation Metrics**: Custom scoring systems for visual cognition tasks

## Research Impact

This work advances our understanding of multimodal AI capabilities and provides a standardized framework for evaluating visual reasoning in large language models. The benchmark has been made available to the research community to facilitate further development in this critical area.
